# E‑Learning nhân vật lịch sử: Trường Chinh

Website e‑learning tương tác, nhiều animation, giới thiệu chi tiết về Trường Chinh (Đặng Xuân Khu) với dòng thời gian, tác phẩm, và quiz ôn tập.

## Chạy dự án

```bash
npm install
npm run dev
```

## Thay ảnh và bổ sung tư liệu

- Ảnh: thêm vào `src/assets/img/` và cập nhật đường dẫn trong `src/data/events.json`.
- Dữ liệu: cập nhật tại `src/data/bio.json`, `src/data/events.json`, `src/data/works.json`.

## Cấu trúc chính

- `src/App.jsx`: bố cục trang, navbar, hero, tiểu sử, timeline, tác phẩm, quiz.
- `src/index.css`: hiệu ứng reveal, modal, timeline, nền hero.
- `src/data/`: dữ liệu nội dung.

## Gợi ý mở rộng

- Thêm ảnh, video, audio tư liệu; nút chia sẻ; đánh dấu mục yêu thích; phân loại/tag sâu hơn.
- Thêm mục "Tranh luận & sửa sai" để trình bày khách quan về cải cách ruộng đất 1953–1956.
- Thêm nguồn tham khảo cuối trang.
